# Release Notes

**v4.2.0** (2024-03-01)

New features and improvements for the Spring release.

- Introduced AI-powered search suggestions
- Added bulk delete for admin users
- Fixed: pagination reset when filters change

---

**v4.1.0** (2024-02-01)

- Added dark mode support for the mobile app
- Performance: reduced initial load time by 30%
- Fixed: email notifications not respecting quiet hours

---

**v4.0.0** (2024-01-15)

Major release with breaking changes.

**Breaking:** The REST API now uses v2 endpoints exclusively. v1 endpoints have been removed. See migration guide at docs.example.com/migrate.

- Complete API redesign with improved consistency
- New SDKs for Python, Go, and Ruby
- Dropped IE11 support
