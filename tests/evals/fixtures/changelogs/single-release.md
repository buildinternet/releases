# v3.2.0 — Dark Mode Support

_Released on January 15, 2024_

We're excited to announce dark mode support across the entire application.

## What's New

- **Dark mode toggle** in the settings panel
- Automatic detection of system-level dark mode preference
- All components now support both light and dark themes

## Bug Fixes

- Fixed sidebar not scrolling on mobile devices
- Resolved an issue where notifications would duplicate on reconnect
