# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Danish translation

## [1.1.1] - 2023-03-05

### Fixed
- Markdown links are now rendered correctly in the preview

## [1.1.0] - 2023-02-12

### Added
- New theme picker with 12 built-in themes
- Custom CSS support for advanced users

### Changed
- Default font changed from Arial to Inter

### Deprecated
- The `legacy-theme` option will be removed in v2.0.0

## [1.0.0] - 2023-01-01

### Added
- Initial stable release
- Markdown editor with live preview
- Export to PDF and HTML
- Plugin API for third-party extensions

### Security
- Fixed XSS vulnerability in markdown renderer
